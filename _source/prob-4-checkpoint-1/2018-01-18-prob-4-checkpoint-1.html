Parte 1

A primeira parte destinada para avaliar uma trilogia e idenfificar qual dos episdios h melhor avaliao e para qual h mais variao nas notas atribudas ao filme.

A trilogia escolhida, composta de 3 filmes, foi Jurassic Park. No grfico abaixo podemos ver como esto distribudas as avaliaes para cada filme. O eixo horizontal representa a nota atribuda e a altura que cada barra alcana no eixo vertical referente a contagem de avaliaes para uma dada nota.

avaliacoes <- read.csv("ratings.csv")

jp_movies <- c(480, 1544, 4638)
jp_names <- c("Jurassic Park", "The Lost World: Jurassic Park", "Jurassic Park III")

jp <- subset(avaliacoes, avaliacoes$movieId == jp_movies[1] | avaliacoes$movieId == jp_movies[2] | avaliacoes$movieId == jp_movies[3])

jp["nome"] <- NA
for(i in 1:3){
  jp$nome[jp$movieId == jp_movies[i]] <- jp_names[i]

ggplot(jp, aes(rating)) + geom_histogram(binwidth = 0.5) + facet_grid(. ~
    nome, scales = "free", space = "free") + labs(x="Nota", y="Quantidade de avaliaes")

Com o objetivo de identificar o filme melhor avaliado, usaremos o processo Bootstrap. Ele consiste em permitir a inferncias a partir de uma amostra, considerando intervalos e confina. Dada uma amostra, elementos so selecionados da mesma aleatoreamente e ento calcula-se a mediana. Esse processo ento repetido diversas vezes.

Para cada filme foi aplicado o processo bootstrap e o valores obtidos das medianas so representados no grfico abaixo. Interpretando, temos que a mediana da notas para Jurassic Park est no intervalo de 3.5-4. De maneira anloga, a mediana de Jurassic Park III estar no intervalo 2-3 e de The Lost Word: Jurassic Park no intervalo 2.5-3.

df <- data.frame()

for(i in jp_movies){
  b = bootstrap(subset(jp, movieId==i) , median(rating))
  median.jp = CI.percentile(b, probs = c(.025, .975))
  df <- data.frame(rbind(df, data.frame(median.jp)))
}

df$nome = c(jp_names[1], jp_names[2], jp_names[3])

df %>% 
  ggplot(aes(x = nome, ymin = X2.5., ymax = X97.5.)) + 
  geom_errorbar(width = .2) + labs(x="Nome do filme", y="Mediana")

Como o filme Jurassic Park o que tem a maior nota em sua faixa de valores e a mesma no compartilhada com nenhum outro filme, podemos afirma que esse filme o que possui melhor avaliao. O mesmo no podemos afirmar para os outros dois, j que, como a faixa de valores pertencente a cada um deles possuem valores em comum, no sabemos ao certo qual o exato valor das medianas, impossibilitando uma rpida comparao visual.

Por outro lado, possvel observar que o filme Jurassic Park III possui uma maior variao em suas notas. Tiramos essa concluso obersevando o tamanho da linha traada e a quantidade de valores que seu intervalo inclui. Ao subtrair seus extremos temos uma diferena de 3 - 2 = 1, enquanto os outros filmes tem ambos uma diferena de 0.5.

Na segunda parte temos o problema para avaliar relacionado a quantidade de gneros que um filme possui. Queremos saber se filmes com uma determinada quantidade de gneros recebem melhores avaliaes.

Para isso, primero reunimos em uma tabela todos os filmes, classificando-os pela quantidade de gneros que cada um possui. Para cada filme tambm calculada a mediana de suas avaliaes. Para esta anlise, descartamos todos os dados que no possuem gneros ou avaliaes. Ou seja, so considerados apenas aqueles com nmero de gneros >= 1 e mediana de avaliaes >= 1.

movie.genre <- read.csv("movie-genre.csv")
